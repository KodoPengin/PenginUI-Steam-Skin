#### Kurzbeschreibung ####
Die beipackte Datei "del_u3a.bat" durchforstet den Standard-Steamordner nach "UnityCrashHandler64.exe" und der "UnityCrashHandler32.exe" und löscht diese aus jeweiligen Spielverzeichnissen.
Dies beugt einer heimlichen Datenaufnahme und Versand von Crashlytics der Firma Unity Technologies, ApS vor und verstärkt die eigene Privatsphäre während der eigenen Spiele-Sessions.
